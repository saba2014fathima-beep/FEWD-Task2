# ParentDiary

**ParentDiary** is a web application designed to help parents and caregivers maintain a structured record of a child’s daily activities. Users can log feeding times, nap durations, and mood changes, and visualize trends on a dashboard. The platform simplifies childcare management and provides an organized way to track a child’s day-to-day development.

---

## Features / Functionalities

- **Log Daily Activities:**  
  - Feeding quantity in ml  
  - Nap duration in hours  
  - Mood tracking (Happy, Neutral, Sad)  

- **Dashboard:**  
  - Displays total feeding and nap hours  
  - Mood summary with counts  
  - Mood trends visualized using a bar chart  

- **Responsive Layout:** Works on both desktop and mobile devices.  

---

## Technology Stack

- **HTML5** – Structure of the application  
- **CSS3** – Styling with pink, blue, and green theme  
- **JavaScript** – Functionality and dynamic updates  
- **Chart.js** – Mood visualization

## Setup Instructions

1. **Download the Project**  
   - Save the HTML code as `index.html`.  

2. **Open in Browser**  
   - Double click `index.html` or open in any browser (Chrome, Edge, Firefox).  

3. **Start Logging Activities**  
   - Enter feeding, nap duration, and select mood  
   - Click **Add Activity** to save  
   - Dashboard updates automatically with totals and chart
